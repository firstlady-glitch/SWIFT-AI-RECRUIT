# **1️⃣ ADMIN (CEO / SUPER ADMIN)**

**Highest-level access. Full control. See everything.**

**The Admin is the CEO and system owner, with unrestricted access to all platform features, settings, analytics, and oversight tools.**

### **A. User Governance**

* **Approve/reject Recruiter registrations**

* **Approve/reject Client (Employer) registrations**

* **Manage ALL users:**

  * **Admins**

  * **Recruiters**

  * **Employers**

  * **Applicants**

* **Suspend, delete, or limit any user**

* **Resolve escalations (candidate ownership, recruiter-client disputes)**

### **B. Job & Recruitment Oversight**

* **Post jobs**  
* **View ALL jobs posted on the platform:**

  * **Jobs created by Clients**

  * **Jobs created by Recruiters**

* **Track job status and activity**

* **See jobs Recruiters are sourcing candidates for**

* **View complete pipelines for all roles**

* **Monitor recruiter performance and employer activity**

### **C. Platform Settings & AI Configuration**

* **Configure AI models (matching algorithms, scoring rules, ranking logic)**

* **Manage system-wide automation rules**

* **Control resume parsing models**

* **Manage global job distribution channels (API, job boards)**

### **D. Subscription, Billing, & Payments**

* **Create, modify, or delete subscription plans for:**

  * **Clients (Employers)**

  * **Recruiters**

  * **Applicants (optional premium tier)**

* **Set feature access per subscription level**

* **Manage payment gateways (Stripe, Paystack, crypto)**

* **View platform-wide transactions & revenue analytics**

* **Issue refunds or credits**

* **View invoices, usage logs, earnings dashboards**

### **E. Security & Compliance**

* **Access full audit logs and system activity trails**

* **Manage API Keys (generate, reset, revoke)**

* **Control roles & permissions (RBAC settings)**

* **Enforce platform-wide security (MFA, IP restrictions)**

* **Manage legal pages (Terms, Privacy, Data Policy)**

### **F. Platform Management** 

* **Manage email/SMS template system**

* **Control notifications, alerts, onboarding messages**

* **Manage integration settings (ATS, HRIS, CRM, webhook configurations)**

* **Override ANY action performed by any user**

* **Export ALL platform data (analytics, user data, job data)**

**The Admin sees everything, controls everything, and governs the entire ecosystem.**
